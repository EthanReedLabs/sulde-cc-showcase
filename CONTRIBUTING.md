# 参与方式

sulde-cc 主源码在 private 主仓库。本 showcase 仓库欢迎以下参与：

---

## 反馈 / Issue

- **Bug 报告**：https://github.com/EthanReedLabs/sulde-cc-showcase/issues
- **Feature Request**：同上 · 加 label `enhancement`
- **方法论讨论**：GitHub Discussion（待开）

---

## 文档贡献

本仓库接受**文档层面的 PR**：

- 修改错别字
- 补充例子
- 改进说明清晰度
- 翻译

代码 PR 因主仓库私有，暂不接受。

---

## Code of Conduct

- 尊重技术讨论
- 不发广告 / 无关内容
